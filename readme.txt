-------------------------------------------------
-                  Lost Echo                    -
-------------------------------------------------

A Global Game Jam 2021 by Aumniez & Iarwain.

Can be played in solo (high score mode,
with random auto-shuffle) or multiplayer.

Frank Brizuela
Romain Killian

-------------------------------------------------

Controls:

Player1
-------
Left Hand               - WASD
Right Hand              - Arrow keys
PickUp / Drop           - Space
Shuffle Drop            - Shift
            .. or ..
Gamepad #1
Left Hand               - Left Stick or D-Pad
Right Hand              - Right Stick or ABXY
PickUp / Drop           - Bumpers
Shuffle Drop            - Triggers

Player2
-------
Gamepad #2

Player3
-------
Gamepad #3

Player4
-------
Gamepad #4

Menu
----
Left / Right            - Select Deck Size
Up / Down               - Select Deck Type
Key 1, 2, 3 or 4        - Select Player(s) Count
            .. or ..
A on Gamepad #1         - 1P
A on Gamepad #2         - 2P
A on Gamepad #3         - 3P
A on Gamepad #4         - 4P

Game Over
---------
Space / A / Back        - Go to Player Select

Misc
----
F12 / Thumb Stick       - Capture a screenshot
